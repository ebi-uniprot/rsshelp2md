UniProt's policy with internet browser support is to support all major browsers up to the oldest version that is also supported by the vendor.

We recommend to use one of the following major web browsers for the UniProt website:

  • FireFox: Choosing a Firefox update channel
  • Chrome: Latest stable release
  • Safari: Look for "Safari" in Apple security updates
  • Edge: Microsoft Edge release notes

We currently also support Internet Explorer. A small and constantly decreasing proportion of our users is using Internet Explorer, and we may therefore phase out support for this browser. See Lifecycle FAQ - Internet Explorer and Edge .

Please note that for older versions of these browsers certain features of the website may not be available (you can check here which browser version you are using).
