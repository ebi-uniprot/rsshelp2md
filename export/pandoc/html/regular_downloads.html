If the data sets you need are available on our ftp site , we recommend that you download them from there. Other data sets can be downloaded by querying this website (read how to retrieve entries via queries programmatically ), but please use a tool that makes use of the HTTP protocol's If-Modified-Since header to avoid more than one download per release.
