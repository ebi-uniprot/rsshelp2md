This subsection of the 'Pathology and Biotech' section is used for proteins that cause an allergic reaction in mammals. We usually specify in which species the protein is allergenic.

Example: Q06811

There is however an exception for an enzyme that does not cause allergic reaction by itself, but catalyzes a modification responsible for multiple allergies in human.
Example: P45669

Related keyword: Allergen
